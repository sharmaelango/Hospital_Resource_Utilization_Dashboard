# Hospital_Resource_Utilization_Dashboard


## Live Deployment

The FastAPI backend for this project is deployed and publicly accessible at:

[https://hospital-resource-utilization-dashboard-1.onrender.com<MY-RENDER-URL>](https://hospital-resource-utilization-dashboard-1.onrender.com)

This backend simulates an on-prem hospital analytics system and serves as the data layer for the Power BI dashboard.  
Due to healthcare data privacy constraints and competition rules, the Power BI dashboard is demonstrated locally.
